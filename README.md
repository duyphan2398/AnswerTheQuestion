# AnswerTheQuestion_NaiveBayesClassifier_KNN
Dữ liệu : thông tin về trường đại học Tôn Đức Thắng gồm 3 file trong thư mực data: questions, answers, labels 
Bài toán : Giải quyết vấn đề phân lớp câu hỏi theo labels, dự đoán câu hỏi thuộc label nào bằng Naive Bayes Classifier.
Tìm câu trả lời cho câu hỏi đó dựa trên data đã có với giải thuật KNN. 

#ToRun: chạy file demoFinal.py =>> python3 demoFinal.py và nhập câu hỏi
